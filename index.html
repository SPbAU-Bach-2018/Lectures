    Полезная информация

    Таблица записи.

    Перед началом работы прочитайте guidelines, они помогут вам верстать так, как мы договорились.

    Чтобы начать работать, установите себе Git каким-нибудь способом:

    • Windows
    • Linux
      • В вашем репозитории есть консольный Git. Например
      • Debian/Ubuntu: # apt-get install git
    • OS X
      • Графическое приложение от GitHub

    Далее, скачайте себе репозиторий через оконное приложение или в командой

    git clone https://github.com/SPbAU-Bach-2018/Lectures.git
