    Установка под Windows

    TeX

    • Установите MikTeX
    • Установите шрифты (скачайте, распакуйте, выберите в Проводнике все файлы и нажмите правой кнопкой Установить):
      • Computer Modern Unicode (cm-unicode)
      • Latin Modern Math (lm-math)
    • Установите из менеджера пакетов (Пуск-&rt;gt;MiKTeX-&rt;gt;Maintainment-&rt;gt;Package Manager) пакет minted. Замените файл (MiKTeX)/tex/latex/minted.tex на этот.

    Всё остальное MikTeX может установить сам ‐ когда он первый раз спросит, снимите галочку и нажмите OK. Верстайте с помощью команды 

    texify --pdf --engine=xetex --tex-option=-8bit --tex-option=-shell-escape <ваш файл&rt;gt;
    Её можно поставить для сборки из Far по нажатию Enter (F9-&rt;gt;Commands-&rt;gt;File Assotiations-&rt;gt;Insert, маска *.tex, в качестве файла укажите !.!). Аналогично, можно настроить эту команду в TeXworks в меню Правка-&rt;gt;Настройка-&rt;gt;Вёрстка.

    Git

    Если вы предпочитаете работу с визуальным приложением, которое, к сожалению, не даст хорошей настраиваемости, но с которым легко работать, ставьте GitHub for Windows. В ином случае, рекомендую связку Far + консольный Git
